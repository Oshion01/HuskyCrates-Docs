Welcome to HuskyCrates documentation!
=====================================================

HuskyCrates is a multipurpose easy to use crate reward system for Sponge.

To familiarize yourself with HuskyCrates check out the `forum page`_ on Sponge.

This plugin has an auto notifier that lets you know when there is a new release available on GitHub_. Please check your console everytime you check your server for new releases.

.. _forum page: https://forums.spongepowered.org/t/package-huskycrates-v1-1-0-beta-api-5-x-7-x-cratesreloaded-but-free-or-something/16433
.. _GitHub: https://github.com/codeHusky/HuskyCrates-Sponge/releases
